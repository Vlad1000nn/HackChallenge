# Инструкция по запуску (Linux)

## 0. Активировать виртуальное окружение (опционально)

0.1. Перейти в директорию ./back
```sh
cd back
```
0.2. Создать виртуальное окружение
```sh
python3 -m venv .venv
```
0.3. Активировать виртуальное окружение
```sh
source .venv/bin/activate
```

## 1. Развернуть приложение
Запустите основной файл проекта находящийся по пути ./back/main.py
```sh
python ./main.py
```

## 2. Изменить параметры файла test.json под входные данные (путь к файлу: tests/test.json)

## 3. Запустить файл теста по пути tests/test.py
```sh
python ../tests/test.py
```
